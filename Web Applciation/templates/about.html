Introduction

The Automated Document Processing Using Deep Learning aims to develop a system that will enable front desk operators to upload scanned business cards, automatically extract relevant entities, and visualise the extracted information. This project incorporates the following techniques to achieve objectives.

  • Optical character recognition.
  • Named entity recognition.
  • Image processing.

Problem Statement

Manually extracting relevant information from business cards can be a time-consuming, error-prone process in the corporate world. By developing a web-based application for extracting and accurately identifying the following entities from scanned business cards:

  • Organisation name.
  • phone number.
  • email addresses.
  • Website address.

Developing a user-friendly web interface for the application will help operators save time and provide an efficient solution for digitizing and managing contact information.

Methodology

The following structural approach will be used for my thesis :

  • Data Collection:
    • Collect diverse Business cards and scan them through mobile or dedicated scanners.
    • Use Pytesseract to extract raw text.
    • Manually label key entities using BIO tagging for supervised learning.
  • Preparing Data to train the model
    • Garbage character removal in whole dataset.
    • Convert labelled data to SpaCy NER training format
    • Split data into training and testing sets for model testing.
  • Model Training :
    • Training Named Entity Recognition model in SpaCy.
    • Evaluate performance.
  • Image Processing :
    • edge detection through OpenCV
  • On-premises web application
    • Usage of Flask for developing a web application.
    • JavaScript integration to create bounding boxes.

Expected outcomes

  • Train a deep learning model to identify entities of a business card.
  • Significant reduction in time and human effort during business card processing.
  • This Foundational project can be extended to invoice-type documents and can also be laterally modified in a facial recognition project.

Expected Findings

  • Integration of OCR, deep learning NER, and image processing will give a positive direction for improvement for traditional-based OCR methods.
  • Overall results will help me to propose an automated reading and scanning system for business cards, which can be used in departments like human resources.
